# 快速入门

## 运行环境

```shell
node v22.15.1
npm v10.9.2
```

## 克隆PipPop文档项目

`git clone https://github.com/SwiftBMan/vuepress-pippop-docs.git`

## 启动项目

```shell
npm run docs:dev
```

## 构建项目

```shell
npm run docs:build
```

## 自动部署到 Github pages

配置自动部署文件路径 .github/workflows/deploy-docs.yml, 已经写好，无需修改
写好md文档直接推送到Github即可，Github 会自动部署

```shell
git add .
git commit -m "更新的内容描述"
git push
```

## 注意事项

无
